Importing edX courses using LaTeX

latex2edx is a preliminary tool which may be used to construct an edX course from a [mathjaxinline]\LaTeX[/mathjaxinline] file.

Requirements

  1. Download and install latex2edx package
  2. Install PlasTeX (on some systems: pip install plastex)
  3. Install python lxml library (on some systems: pip install lxml)
  4. Install xmllint (on Ubuntu, apt-get install libxml2-utils)

Hints and Guidelines

  1. latex2edx can be used to create entire courses from one latex file, to update problems in (or add problems to) an existing course, or to simply generate individual problem XML files from a latex file.
  2. Do not put spaces around the "=" sign in the abox statements; the abox parameters are space delimited.
  3. Each problem should be given a unique name; don't reuse "Problem 1" in different files, for example. This is because each problem's filename is generated from the name.
  4. Under osX, generate the tex files using emacs or vi instead of textedit. Latex cannot take special characters in unicode format, which are being generated by textedit.
  5. Latex errors (mismatched tags, typos, etc) can be caught by running latex or pdflatex on the file. Use the edXpsl.sty style file, and latex should produce a fine user-readable output file.
  6. The edX problem css stylesheets currently do not allow inline display of answer input boxes. This is being fixed.

Example

Here is a simple example of a latex file from which a rudimentary course can be generated:

\documentclass[12pt]{article}

\usepackage{edXpsl}	% edX

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\begin{document}

\begin{edXcourse}{1.00x}{1.00x Fall 2012}

\begin{edXchapter}{Unit 1}

\begin{edXsection}{Introduction}

\begin{edXsequential}


%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\begin{edXproblem}{Problem 1}{40}

This is a sample problem.  What is the number made famous by
Hitchhiker's guide to the galaxy?

\edXabox{type='symbolic' size='90' expect='42' }

\end{edXproblem}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\end{edXsequential}

\end{edXsection}
\end{edXchapter}
\end{edXcourse}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\end{document}
Note: Images (via includegraphics) can be made to work; YMMV.
